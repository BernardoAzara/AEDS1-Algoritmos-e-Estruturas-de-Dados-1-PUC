# AEDS1-Algoritmos-e-Estruturas-de-Dados-1-PUC
Repositório com exercícios, trabalhos e projetos da disciplina de Algoritmos e Estruturas de Dados I (AEDS1), desenvolvidos em C. Abrange lógica de programação, funções, recursão, resolução de problemas computacionais e etc. O objetivo deste repositório é documentar a evolução do aprendizado e consolidar conhecimentos práticos.
